# Automation — Instagram

Implementado: publicação de carrossel via Meta Graph API
(`publish_instagram.py`). Fluxo completo (pré-requisitos, setup de token,
publicação, troubleshooting) e checklist vivem na skill `instagram` — não
duplicados aqui.

## Conteúdo desta pasta

- `publish_instagram.py` — script de publicação.
- `.env.example` — variáveis necessárias (`INSTAGRAM_BUSINESS_ID`,
  `FACEBOOK_PAGE_ID`, `INSTAGRAM_ACCESS_TOKEN`). Copiar para `.env` e
  preencher — nunca versionar o `.env` real.

Para setup, uso e resolução de erro: skill `instagram`.
