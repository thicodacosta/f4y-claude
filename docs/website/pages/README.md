## Regra de reuso

- **Copy**: nunca inventar copy de serviço — basear em `docs/servicos/` e no
  tom de `docs/marca/tom-de-voz.md`.
- **Componentes**: nunca inventar componente novo aqui — usar o que existe em
  `design-system/components/`; se faltar, ele nasce lá primeiro.
- **SEO**: aplicar o checklist on-page da skill `seo`.

Nenhuma página abaixo está totalmente especificada — campos pendentes ficam
`[TODO] Definir com Thiago`.
