# Templates de Descrição de Vaga

Usado pela skill `recrutamento`. Dois templates, um por vertical — a estrutura
é a mesma, o que muda é o vocabulário e os critérios de avaliação (ver tabela em
`docs/metodologia/recrutamento.md`):

- `tecnologia.md` — vagas de Tecnologia
- `corporativo.md` — vagas de Corporativo

## Como usar

1. Parta do perfil de vaga já validado com o cliente (Fase 1 da metodologia,
   `docs/metodologia/recrutamento.md`) — não redija a descrição antes de o
   perfil estar confirmado.
2. Preencha os campos entre `[colchetes]` do template correspondente à vertical.
3. Aplique o tom de voz descrito em `docs/marca/tom-de-voz.md` — as instruções
   em *itálico* dentro do template lembram o pilar relevante em cada seção,
   mas não substituem a leitura do documento completo.
4. Rode o checklist ao final do template antes de considerar a vaga pronta
   para publicar.

## Estrutura comum às duas verticais

Ambos os templates seguem as mesmas seções, na mesma ordem, para manter
consistência de marca entre vagas de qualquer área:

1. Título da vaga
2. Sobre a oportunidade (contexto de negócio, não só o cargo)
3. Responsabilidades
4. Requisitos obrigatórios
5. Requisitos desejáveis
6. O que oferecemos
7. Processo seletivo — etapas padrão em `docs/processos/processo-seletivo.md`,
   não redigidas do zero a cada vaga
