# Templates de Instagram

Usado pela skill `instagram`. A pauta já deve estar definida (pela skill
`marketing-conteudo` ou diretamente pelo usuário) antes de usar estes templates
— eles cobrem o formato, não a escolha do tema.

- `carrossel.md` — carrossel de 2 a 10 slides + legenda (formato padrão do canal)

## Como usar

1. Confirme a pauta e a vertical de negócio em foco antes de preencher.
2. Slides seguem os componentes visuais definidos no Design System (skill
   `design-system`) — nunca criar layout novo de slide sem checar se já existe
   um padrão equivalente.
3. Aplique o tom de `docs/marca/tom-de-voz.md` na legenda e no texto de cada slide.
4. Rode o checklist final da skill `instagram` antes de considerar pronto para
   exportar/publicar.
