# Templates de Landing Page

Usado pelas skills `website` e `lovable` para copy de páginas de serviço do
site institucional. Cobre o texto — componentes visuais vêm sempre de
`design-system/` (skill `design-system`), nunca criados ad-hoc aqui.

- `pagina-servico.md` — estrutura de copy para a página de uma vertical de negócio

## Como usar

1. Confirme a vertical antes de preencher — o conteúdo factual do serviço vem
   de `docs/servicos/` (`recrutamento.md`, `executive-search.md` ou
   `alocacao-tech.md`), nunca reescrito do zero.
2. Aplique o tom de `docs/marca/tom-de-voz.md`.
3. Estruture os blocos em componentes já existentes no Design System — se um
   componente necessário não existir, ele nasce na skill `design-system` antes
   de ser usado aqui.
4. Antes de publicar no site, rode o checklist on-page da skill `seo`.
