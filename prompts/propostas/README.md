# Templates de Proposta Comercial

Usado pela skill `propostas-comerciais`. Três templates, um por vertical de
origem (Recrutamento cobre Tecnologia e Corporativo com o mesmo template — ver
`docs/metodologia/recrutamento.md`):

- `recrutamento.md` — Tecnologia e Corporativo
- `executive-search.md` — posições C-level/board
- `alocacao-tech.md` — staffing

## Estrutura comum às três verticais

Todos seguem as mesmas 5 seções, na ordem definida pela skill
`propostas-comerciais` — o que muda entre os templates é o vocabulário e o
foco de argumento de cada vertical, nunca a estrutura:

1. Contexto e entendimento da necessidade do cliente
2. Escopo do serviço
3. Metodologia resumida
4. Investimento e condições comerciais
5. Prazos e próximos passos

## Como usar

1. Confirme a vertical correta antes de escolher o template — a estrutura
   comercial (fee vs. rate) muda entre recrutamento/executive search e
   alocação tech (ver `docs/processos/fluxo-comercial.md`).
2. Preencha os campos entre `[colchetes]`.
3. Para a seção de metodologia, resuma — não copie o documento inteiro de
   `docs/metodologia/`. Só o relevante para esta proposta.
4. Para condições comerciais, use os valores reais acordados com o cliente;
   os números nos templates são placeholders ilustrativos.
5. Rode o checklist de revisão da skill `propostas-comerciais` antes de enviar.
