# Templates de LinkedIn

Usado pela skill `linkedin`. A pauta já deve estar definida (pela skill
`marketing-conteudo` ou diretamente pelo usuário) antes de usar estes templates.

- `post-empresa.md` — fala em nome da Find4You (página da empresa)
- `post-pessoal.md` — thought-leadership na voz de um profissional da Find4You

## Como usar

1. Confirme qual dos dois formatos antes de redigir, se não estiver explícito
   no pedido — os templates não são intercambiáveis (ver skill `linkedin`).
2. Aplique o tom de `docs/marca/tom-de-voz.md`.
3. Rode o checklist final da skill `linkedin` antes de publicar.
