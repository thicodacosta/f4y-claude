# Templates de Artigo/Blog

Usado pela skill `marketing-conteudo` para conteúdo longo-form original. A pauta
já deve estar definida (decisão de "o quê" e "por quê") antes de usar este
template — ele cobre estrutura e formato, não a escolha do tema.

- `artigo.md` — estrutura padrão de artigo/blog

## Como usar

1. Confirme a pauta e o objetivo de negócio por trás dela (skill
   `marketing-conteudo`) antes de redigir.
2. Aplique o tom de `docs/marca/tom-de-voz.md`.
3. Ao publicar no site, rode o checklist on-page da skill `seo`.
4. Artigos podem depois ser adaptados para posts de canal pelas skills
   `instagram`/`linkedin` — o artigo nasce aqui como peça original.
